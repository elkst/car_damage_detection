Название проекта: Car Damage Detection & Classification System

Описание: Веб-приложение для автоматического детектирования, классификации и анализа повреждений кузова автомобиля на основе модели YOLOv8. Система предоставляет пользовательский интерфейс для загрузки изображений, визуализации результатов с bounding boxes и детальной статистики (площади повреждений, уровень серьёзности ущерба).

Ключевые технологии:

Модель: Ultralytics YOLOv8 (обучение и инференс)

Бэкенд: FastAPI (Python)

Фронтенд: Jinja2 (HTML/CSS)

Обработка изображений: Pillow (PIL)

Датасет: Automobile Damage Detection (Roboflow Universe)

Структура репозитория (пример):

train.py — скрипт для обучения модели YOLOv8.

main.py — основной файл FastAPI-приложения.

templates/index.html — шаблон веб-интерфейса.

static/ — стили, загруженные и обработанные изображения.

requirements.txt — файл зависимостей Python.

best.pt / best_car_damage_model.pt — веса обученной модели.

data.yaml — конфигурация датасета.
